    背景： 1600s起源于罗马，随后传播到欧洲的大部分地区。17世纪是贵族掌权的世纪。伏尔泰称17世纪是“路易十四的世纪”。科学史上，怀特海称之为“天才的世纪”。 科学在伽利略、牛顿等人的开创下展开。资产阶级兴起，君主政治渐独立于宗教之外(但民主思想萌芽)。文化上“除旧布新”，在各方面都有重大的改变与成就。艺术上“世俗化”，精力充沛，勇于创新，甚至好大喜功。文艺上的名家如莎士比亚、塞万提斯；绘画上有鲁本斯，伦勃朗等，影响都极深远。也在这时期，欧洲向外扩张殖民，渐渐占据了世界文明的中心。
    巴洛克的流行与罗马天主教会的鼓励有关。为了回应宗教改革，教会决定艺术应当直接地充满感情地表达宗教主题。贵族认为巴洛克是表达胜利、权利和控制的手段。带有前庭、大楼梯和豪华的会客室的巴洛克宫殿为之兴建起来。
    词源： 欧洲人最初用这个词指“缺乏古典均衡特性的作品”，是对17世纪不同于文艺复兴风格的一个带贬义的称呼。
    特点： 运用夸张的运动性和清晰的细节来营造戏剧、紧张、繁琐、恢宏的效果。

    《第94交响曲》 (惊愕/Surprise) #

    海顿的《惊愕交响曲》作于1791年，传说当时伦敦的贵族是音乐会的常客，但是他们来听海顿的音乐会只是为了表现自己所谓的高雅品位，在那里附庸风雅，每每在乐队演奏时打瞌睡。海顿知道后非常生气，于是他就写了这部《惊愕交响曲》。
    1.Adagio-Vivace assai(柔板—活泼的很快的快板)
    奏鸣曲式。从一段如歌的柔板开始，由木管乐器和弦乐器分别奏出，然后浪漫派似的渐弱。主部主题是舞曲般的音型和匈牙利吉普赛的曲调，由小提琴奏出。副部主题并不具备旋律性，只是弦乐上的节奏音型。发展部非常简练。整个乐章的性格是妩媚优雅而不深刻。
    2.Andante(行板)
    Movement II
    海顿交响曲中最受欢迎的乐章。主题用脍炙人口的古老的法国流行儿歌“妈妈，请听我说”，现代英语儿歌《小星星》也采用此旋律）的变体，乐队突然爆发出一个响亮的和弦，让人猛地惊醒，这就是“惊愕”。这一乐章的曲式是主题和四段变奏。
    3.Menuetto：Allegro molto(小步舞曲：很快的快板)
    小步舞曲的速度极快，嬉戏幽默的性格使它完全不像宫廷式的小步舞曲。
    4.Finale：Allegro di molto(终曲：很快的快板)
    回旋奏鸣曲式，海顿喜欢把回旋曲的华丽活泼与奏鸣曲的复杂变化结合在一起。乐章以两个展翅飞翔，散发着活力的主题为基础，进行速度很迅猛。对于海顿时代的提琴演奏技术，这个乐章的难度很大，即使现在也不容易。

    Wagner 瓦格纳 1813-1883 德国 #


    德国作曲家，以其歌剧闻名。理查德·瓦格纳不同于其他的歌剧作者，他不但作曲，还自己编写歌剧剧本。他是德国歌剧史上一位举足轻重的人物。承接莫扎特、贝多芬的歌剧传统，开启了后浪漫主义歌剧作曲潮流，理查德·施特劳斯紧随其后。

    每个流派三个人 #

    缔造者，奠基者，班长，xx之父。
    经营者，学习委员，只负责自己学习好，稳定高品质。
    改革者，不断创新和挑战权威，风格多变，独树一帜。

    总体上 #

    缔造者： 巴赫 Bach (音乐之父)(巴洛克时期，把音乐允许的表现的乐器和曲式都写了一遍)虔诚的路德派教徒。一千多部作品，数量大，质量高。
    经营者： 莫扎特 Mozart 天才，草稿和完稿一样。虽然一生一千多首，但都是套路，早年晚年没大区别。7岁第一部交响曲，12岁第一部歌剧。垂直作曲。
    改革者: 贝多芬 Beethoven，29岁才写出第一部像样作品。维漂，拜海顿为师。由于早年生活不幸，音乐风格充满力量。出版作品只有135首，但是每一首都在创新和挑战权威。

    印象主义(绘画) #

    印象主义音乐是19世纪(1860s开始)主要在法国流行的一种音乐样式，受象徵主义文学和印象主义绘画的影响。
    马奈，缔造者，真正意义上的第一个印象派画家。
    莫奈，经营者，每一副质量都很高，但是一生没什么变化。
    梵高，改革者，从早期到末期，风格一直在变化，一直在探索，夸张的风格推动了野兽派的诞生。

    印象派作曲家 #

    主要是德布西, 拉威尔 Ravel。另外还有：奥托里诺·雷斯庇基 Respighi 意大利, 米约 Milhaud 法国, 莉莉·布朗热 Lili Boulanger, 普契尼 Puccini 意大利, 拉赫曼尼诺夫 Rachmaninoff, 雷夫·佛汉·威廉斯 Ralph Vaughan Williams 英国, 史克里亚宾 Scriabin 俄国。

    Final Notes #

    历史上以作曲为职业，有出版过作品的，有15k人。总共1百万部作品(一部约几十首)。现在仍在演奏的大约1k首，经常演奏的200首。大部分的古典音乐是为宗教服务，尤其是弥撒、安魂曲，所以比较高冷。但是也有很多是为了宫廷人员下午茶休息听，所以非常休闲舒服。还有就是由于古典音乐没有歌词，没有场景感，很抽象，加之时间久远，所以很难理解。
